Understanding Token Costs

API usage is measured in tokens. A token is approximately 4 characters or 0.75 words in English. Understanding token consumption helps manage costs:

  • Input tokens: The text you send to the AI (prompts, uploaded content)
  • Output tokens: The text the AI generates in response
  • Image tokens: Different pricing applies for DALL-E and image analysis
  • Audio tokens: TTS is priced per character, Whisper per minute
Cost-Saving Tip: Start with smaller, more specific prompts. The GPT-4o Mini models offer excellent results at lower cost compared to larger models.

Tips & Best Practices

Prompt Writing Guidelines

Be Specific: Instead of "create an image of a dog," try "create a photorealistic image of a golden retriever puppy playing in autumn leaves, golden hour lighting, shallow depth of field."
Provide Context: Help the AI understand the purpose. "Write a blog post about climate change for a general audience aged 25-40" yields better results than just "write about climate change."
Use Examples: When possible, show the AI what you want: "Write in the style of this example: [paste example text]"

The 5 W's Framework

Structure prompts using Who, What, When, Where, Why:

  • Who: Target audience or character perspective
  • What: Specific content or action desired
  • When: Time period, season, or temporal context
  • Where: Setting, location, or environment
  • Why: Purpose, goal, or intended effect

Image Generation Best Practices

Effective Prompt Structure

[Subject] + [Action/Pose] + [Environment] + [Lighting] + [Style] + [Quality Boosters]

Example:
"Elegant ballerina, mid-pirouette, grand theater stage, 
spotlight from above, oil painting style, highly detailed, 8k resolution"

Quality Boosters

These terms consistently improve image quality:

  • Highly detailed, intricate details
  • Professional photography, 8k resolution
  • Sharp focus, depth of field
  • Cinematic lighting, volumetric light
  • Masterpiece, award-winning

Aspect Ratios and Sizes

Different use cases require different image dimensions:

  • 1:1 (Square): Social media posts, profile pictures
  • 16:9 (Widescreen): Presentations, banners, headers
  • 9:16 (Portrait): Mobile content, stories
  • 4:3 (Standard): Documents, general use

Cost Management Strategies

Monitor Usage: Regularly check your API dashboards to track spending and set up billing alerts with each provider.

Support & Resources

Troubleshooting Common Issues

Issue Possible Cause Solution
"API Key Invalid" error Key not saved or expired Re-enter API keys in the setup modal. Verify keys are active in provider dashboards.
Images not generating Rate limit or quota exceeded Check OpenAI usage dashboard. Wait a moment and try again with a simpler prompt.
File uploads failing Missing upload directory Ensure ./upload folder exists at project root with write permissions.
Slow response times High API load or large prompts Try again later or break large requests into smaller chunks.
Chat history not saving Local storage disabled or full Enable local storage in browser settings or clear old data.
Audio playback issues Browser autoplay restrictions Click to interact with page before playback. Check volume settings.
